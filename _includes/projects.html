Projects

Blend - Developed a RESTful application to allow communities to borrow and lend items over a secure platform. Blend is built on an Node.js Express backend server, operating off a MySql database. Users can log-in to the application, search for items curated to their preferences on location, request to borrow an item, and rate the service and transaction.
Serengeti Eye - Trained a convolutional network to achieve above 80% accuracy in the classification of African Serengeti wild animals. Minimized overfitting of data by optimizing the C hyperparameter.
mediumware - The mediumware ecosystem project aims to bring programming to anyone. Using this platform which interfaces the intuitive programming platform, Scratch, users can perform a variety of tasks such as sound activated lights, programmatic alarm clocks, etc - all through Scratch. mediumware is the middleman that makes this possible.
Maps - Maps is a rendition of Google Maps localized for Providence, Rhode Island. Features include autocorrected and suggested listings of addresses, shortest paths optimizations using Dijkstra's algorithm and A*, and nearest neighbor search for a user-friendly interface.
Chatroom - Minimalist chatroom web application implemented in NodeJs.
